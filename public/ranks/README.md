# Rank Images — League of Legends

Pasta para os emblemas de rank do LoL. Coloque aqui as imagens com os nomes exatos abaixo.

## Arquivos esperados

| Arquivo          | Rank        |
|------------------|-------------|
| `iron.png`       | Ferro       |
| `bronze.png`     | Bronze      |
| `silver.png`     | Prata       |
| `gold.png`       | Ouro        |
| `platinum.png`   | Platina     |
| `emerald.png`    | Esmeralda   |
| `diamond.png`    | Diamante    |
| `master.png`     | Mestre      |
| `grandmaster.png`| Grão-mestre |
| `challenger.png` | Desafiante  |

## Formato recomendado

- Formato: **PNG** com fundo transparente, ou **WebP**
- Tamanho ideal: **128×128 px** ou **256×256 px**
- Nomes de arquivo: tudo em **minúsculo**, sem espaços

## Onde encontrar

As imagens oficiais de rank do League of Legends estão disponíveis no Data Dragon (CDN da Riot Games).
Pesquise por "League of Legends rank emblems png transparent" ou acesse:

  Riot Data Dragon → ranked-emblems

Os nomes no Data Dragon são no padrão `Emblem_Iron.png`, `Emblem_Bronze.png`, etc.
Baixe e renomeie para os nomes da tabela acima antes de colocar aqui.

Enquanto as imagens não estiverem presentes, o componente exibe automaticamente
ícones de fallback (Lucide Icons com as cores de cada rank).
